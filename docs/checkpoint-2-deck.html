The problem

Agents need more than an opaque API response.

What data?

Public snapshots change. A later reviewer needs to know the exact source, timestamp and snapshot hash.

What rules ran?

Visible-depth and freshness checks must be explicit, deterministic and versioned.

What was delivered?

An agent-to-agent result needs a bounded deliverable, not an unverifiable claim.

OutcomeRail addresses analysis provenance; it does not place or recommend trades.
